jQuery Interview Questions and Answers

  • 1. What is C#?

    C# is an object-oriented programming language developed by Microsoft. It is designed for developing apps that run on the .NET framework.

  • 2. What is the difference between `public`, `private`, `protected`, and `internal` access modifiers?

    `public`: Accessible from anywhere; `private`: Accessible only within the class; `protected`: Accessible within the class and by derived class instances; `internal`: Accessible only within the same assembly.

  • 3. What is an abstract class in C#?

    An abstract class is a class that cannot be instantiated and is meant to be inherited by other classes. It can contain abstract methods with no implementation.

  • 4. What is the difference between an interface and an abstract class?

    An interface contains only method declarations with no implementations. An abstract class can have both abstract and non-abstract methods.

  • 5. What is polymorphism in C#?

    Polymorphism allows methods to have different behaviors based on the object that invokes them. It is achieved through method overloading and overriding.

  • 6. What is encapsulation?

    Encapsulation is the concept of bundling data (variables) and methods (functions) that operate on the data into a single unit, typically a class. It also restricts access to certain components using access modifiers.

  • 7. What are delegates in C#?

    A delegate is a type that represents references to methods. It is used to pass methods as arguments to other methods.

  • 8. What is the difference between `String` and `StringBuilder` in C#?

    `String` is immutable, meaning once created, its value cannot be changed. `StringBuilder` is mutable and allows modification without creating a new instance.

  • 9. What is LINQ?

    LINQ (Language-Integrated Query) is a feature that allows querying of collections and databases in a readable, SQL-like syntax, integrated directly into C# code.

  • 10. What is the `async` and `await` keyword used for in C#?

    The `async` keyword defines an asynchronous method, and `await` is used to pause the method execution until the awaited task completes, making it easy to write asynchronous code.

  • 11. Explain the difference between `==` and `Equals()` in C#.

    The `==` operator compares object references, while the `Equals()` method compares the values of the objects.

  • 12. What is the difference between `throw` and `throw ex` in exception handling?

    `throw` rethrows the original exception, preserving the stack trace. `throw ex` resets the stack trace, which makes it harder to debug.

  • 13. What is a `sealed` class in C#?

    A `sealed` class cannot be inherited. It is used when you want to prevent other classes from inheriting its functionality.

  • 14. What are extension methods in C#?

    Extension methods allow you to add new methods to existing types without modifying them. They are defined as static methods but are called as if they are instance methods.

  • 15. What is dependency injection?

    Dependency injection is a design pattern used to achieve Inversion of Control (IoC) by passing dependencies into an object, instead of the object creating them itself, making the code more modular and testable.

  • 16. What is a constructor in C#?

    A constructor is a special method that is automatically called when an object is created. It is used to initialize the objectâ€™s properties.

  • 17. What is garbage collection in C#?

    Garbage collection in C# is the process of automatically reclaiming memory that is no longer in use, freeing up resources and optimizing memory management.

  • 18. What are the types of inheritance supported by C#?

    C# supports single inheritance, multiple inheritance through interfaces, multilevel inheritance, and hierarchical inheritance.
